# Bot Validador de CPF

Bot desenvolvido em Kotlin utilizando a API do Telegram para validar números de CPF.

## Funcionalidades

- Validação automática de CPF
- Aceita CPF com ou sem formatação
- Resposta instantânea no Telegram
- Integração com API oficial do Telegram


## Como usar

1. Inicie o bot
2. Envie um CPF no chat
3. O bot informará se o CPF é válido ou inválido

Exemplo:

/validar 52998224725

Resposta:

CPF 529.982.247-25 é VÁLIDO.
